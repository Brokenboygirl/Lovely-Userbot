## VPS/Locally deploy!
```console
Lovely@Debian~ $ apt-get -y update
Lovely@Debian~ $ apt-get -y install git gcc python3-dev
Lovely@Debian~ $ git clone https://github.com/Brokenboygirl/Lovely-Userbot
Lovely@Debian~ $ cd Lovely-USERBOT
Lovely@Debian~ $ pip3 install -U-r requirements.txt
Lovely@Debian~ $ cp sample.env .env
```

<h3 align="center">
